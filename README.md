# 🌍 AI Voice Translator

## About the Project

AI Voice Translator is a Python desktop application that helps users translate speech from one language to another in real time.

The application converts voice into text, translates it into the selected language, and also speaks the translated text. This project was developed as a learning project and is the first step towards building a real-time multilingual calling application.

---

## Features

- 🎤 Speech to Text
- 🌐 Real-Time Language Translation
- 🔊 Text to Speech
- 🌍 Supports Multiple Languages
- 🔄 Swap Languages
- 📜 Conversation History
- 💾 Save Conversation
- 📋 Copy Translation
- 🗑️ Clear Conversation
- 🟢 Conversation Mode
- 🎨 Simple and Modern User Interface

## How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/ShravaniKarpe/AI-Voice-Translator.git
```

### 2. Open the Project Folder

```bash
cd AI-Voice-Translator
```

### 3. Install Required Libraries

```bash
pip install -r requirements.txt
```

### 4. Run the Project

```bash
python main.py
```
