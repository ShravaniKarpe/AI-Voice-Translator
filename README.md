# 🌍 AI Voice Translator

A desktop application built using Python that converts speech from one language into another language using AI.

## Features

- 🎤 Speech Recognition
- 🌍 Language Translation
- 🔊 Text-to-Speech
- 🔄 Language Swap
- 🖥️ CustomTkinter GUI

## Technologies Used

- Python
- CustomTkinter
- SpeechRecognition
- Deep Translator
- gTTS
- Pygame

## Supported Languages

- English
- Marathi
- Hindi
- Tamil
- Telugu
