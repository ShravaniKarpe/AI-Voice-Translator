# 🌍 AI Voice Translator

An AI-powered desktop application that enables real-time voice translation between multiple languages.

## ✨ Features

- 🎤 Speech Recognition
- 🌍 Language Translation
- 🔊 Text-to-Speech
- 🔄 Language Swap
- 💬 Conversation Mode
- 🟢 Professional Status Bar
- 🖥️ Modern CustomTkinter Interface
